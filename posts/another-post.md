# Welcome to My Blog yeet yeet

This is an example blog post written in markdown. You can edit this file or create new ones in the `/posts/` directory.

## Features

- **Markdown Support**: Write your posts in markdown
- **Frontmatter**: Add metadata like title, date, and description
- **Simple CMS**: Just edit markdown files and push to deploy

## Getting Started

To create a new blog post:

1. Create a new `.md` file in the `/posts/` directory
2. Add frontmatter at the top with title, date, and description
3. Write your content in markdown below the frontmatter
4. The file name will become the URL slug (e.g., `my-post.md` â†’ `/posts/my-post`)
